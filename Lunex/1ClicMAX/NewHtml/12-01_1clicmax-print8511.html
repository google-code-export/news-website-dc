Dialing Instructions for 1ClicMAX™
MAKE A PINLESS CALL IN 2 STEPS
  • Step 1: Dial Access Number (either local or toll free)
  • Step 2: Dial 011 + country code + area code+ phone number + # key
Note:
  • Please check with your telephone provider to make sure that the Local Access Number you dial is LOCAL to your phone service to avoid paying long distance charges.
  • DO NOT press TALK/SEND after dialing phone numbers when using mobiles.
  • When using 1-888-991-5959 Toll-Free Number, additional two cents per minute will apply.
  • When using Alaska access number 1-888-777-0139, additional twenty cents per minute will apply.

SPEED DIALING
Set up Speed Dial & Enjoy 2Step-Calling
  • Step 1: Press assigned speed dial number for access number.
  • Step 2: Press assigned speed dial for international phone number.

1Clic™ DIALING
  • Dial 1Clic™ Number, press Talk/Send, call will be connected.
Note:
  • The 1Clic™ Number can be dialed directly from phone book.
  • The 1Clic™ Number can be set up as speed dial on the handset. After that, just 1 touch to the assigned speed dial number and hold for a few seconds, call will be connected. This function operates differently depending on handset.
